# PRD v1.2 最终发布审计与真机验证

## 背景

用户要求在 PRD v1.2 开发后完成全局 review 和真机测试。核心功能与导航收尾已实现；本任务用于在最后一次代码变更后重新建立发布证据，避免沿用变更前的 APK/设备测试结果。

## 验收范围

- 检查 PRD v1.2 已归档实现任务与导航收尾任务记录。
- 重新执行当前代码的 Flutter 静态检查与测试。
- 重新构建 release APK。
- 在可用 Android 真机上重新安装当前 release APK。
- 运行覆盖 v1.2 主流程的真机 integration test（如设备可用）。
- 记录 review/验证证据，并归档本 CCG 任务。

## 限制

- 根目录不是 git 仓库，归档 commit 可能不可执行；若无法提交，必须记录原因。
