# Requirements

用户目标：检查 app 首页村列表为什么每条记录都会突出红色部分；先截图首页图片并分析图片；村、楼栋、房源记录都有该问题；修复问题并优化 UI。

## Concrete requirements

1. 截取并保存首页问题截图。
2. 分析截图，确认红色突出块的位置/像素特征与根因。
3. 修复共享滑动删除组件，使村、楼栋、房源记录在未滑动状态不再露出风险红色删除背景。
4. 保留滑动后固定宽度露出删除区、点击“删除”再确认的既有交互。
5. 补回归测试，证明未滑动不露红、滑动后仍显示删除区。
6. 运行静态分析和测试。
7. 构建并安装 release APK 后在真机截图验证：首页、楼栋页、房源列表无默认红色突出块。
8. 将本次踩坑沉淀到 `.ccg/spec/guides/index.md`。
