## 1. 自动化命令

| 命令 | 覆盖重点 |
| --- | --- |
| `flutter test test/data/village_repository_test.dart --reporter expanded` | 删除楼栋/村的级联清理、统计刷新 |
| `flutter test test/features/scan_map_page_test.dart --reporter expanded` | 快速记录校验、楼栋选择/新建、滑动删除、无地图文案 |
| `flutter test` | 全量 unit/widget 回归 |
| `flutter test -d <device> integration_test/full_device_flow_test.dart` | Android 主流程：村→楼栋→快速记录→详情→Checklist→风险→列表→对比 |
| `flutter test -d <device> integration_test/v1_1_formal_device_flow_test.dart` | v1.2 补充真机覆盖：缺失村、多入口、楼栋创建、筛选与对比边界 |

> 文件名 `v1_1_formal_device_flow_test.dart` 为历史命名；脚本内容已更新为 v1.2 语义。

## 6. 发布阻断口径

以下任一失败应阻断发布：

- 进入快速记录即创建空房源。
- 必填不完整仍能保存房源。
- 村级输入新楼栋未创建/未绑定楼栋。
- 楼栋入口保存后丢失 `buildingId`。
- 删除村/楼栋后留下可见孤儿房源。
- 首页主流程出现地图/定位失败阻断文案。
- 自动化 P0 用例失败或未说明无法执行原因。
